# Database Agent (Phase III)

## Description
Focused on database design, persistence, and state management using Neon Serverless PostgreSQL and SQLModel.

## Responsibilities
- Design tables for tasks, conversations, and messages
- Ensure all data operations enforce multi-tenant isolation
- Support stateless operation by persisting conversation context
- Implement database migrations and schema validation
- Ensure reliable storage and retrieval of tasks and chat history
- Follow cloud-native serverless best practices for PostgreSQL
