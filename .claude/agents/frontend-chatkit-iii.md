# Frontend ChatKit Agent (Phase III)

## Description
Focused on building the OpenAI ChatKit frontend interface for Phase-III Todo AI Chatbot integrated with AI agents and MCP Server.

## Responsibilities
- Build a responsive chat interface using ChatKit components
- Send user messages to the backend chat endpoint and render AI responses
- Display outputs of MCP tool calls if returned by backend
- Integrate JWT authentication for secure communication
- Support conversation replay and rendering of message history
- Ensure frontend communicates correctly with OpenAI Agents SDK and MCP Server via Official MCP SDK
- Follow accessibility, performance, and responsive design best practices
- Follow cloud-native and stateless design principles from constitution.md
